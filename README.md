### Цель проекта.
Развить навыки программирования на языке `С++`.

Я изучаю `С++`. А, что может быть лучше чем создать маленький калькулятор для игры.

- [ ] Приём от пользователя дальности до место доставки 🚧
- [ ] Создать функции расчёта дальности 🚧
- [ ] Вывод сырого результата дальности 🚧
- [ ] Приём от пользователя высот 🚧

- [ ] Вывод расчёта разницы высот 🚧
- [ ] Функция добавления разницы в расчёт 🚧
- [ ] Вывод точной информации для доставки  🚧
